Basejump: Image Search Abstraction Layer

  • Given a query string, this api searches the web for relevant images and responds with an array of objects. Each object contains the url of an image, a snippet about the image, its context, and a thumbnail version.
  • By appending ?offset=num to to an api call, where num is a positive number, you can paginate through the responses.
  • Visit here for a list of the latest search strings.
